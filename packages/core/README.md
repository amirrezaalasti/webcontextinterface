# @webcontextinterface/core

Complete WCI SDK — re-exports `@webcontextinterface/spec`, `distiller`, `bridge`, and `context`.

```bash
npm install @webcontextinterface/core
```

```typescript
import {
  WciDistiller,
  WciBridge,
  WciContextLoader,
  readWciNodeSpec,
} from '@webcontextinterface/core';
```
