# @wci/core

Complete WCI SDK — re-exports `@wci/spec`, `distiller`, `bridge`, and `context`.

```bash
npm install @wci/core
```

```typescript
import {
  WciDistiller,
  WciBridge,
  WciContextLoader,
  readWciNodeSpec,
} from '@wci/core';
```
